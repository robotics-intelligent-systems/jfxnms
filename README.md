# Artificial Intelligence for Network Management System

Harvester is a modern, open, interoperable, hyperconverged infrastructure (HCI) solution. Machine Learning in OMNeT++. Artery enables V2X simulations based on ETSI ITS-G5 protocols. AVENS (Aerial VEhicle Network Simulator). space_veins is an extension for Veins simulating Vehicle-to-Satellite Communication. Network-of-Systems Simulator (NoSSim). Gradys Simulations MAVSIMNET a simulation framework for the OMNET++ discrete event simulator. CrowNet is an open source simulation framework for the development of new networked mobility concepts and intelligent transportation systems. Automated Network Simulation and Analysis (ANSA). JetKVM is a high-performance, open-source KVM over IP. Ralph is the CMDB / Asset Management system for data center and back office hardware. SONiC is a free and open-source network operating system. OpenWISP is an open-source solution for efficient IT network deployment, monitoring & management. NMS PRIME is an Open Source Network Provisioning Tool and Network Management Platform. ONOS is the only SDN controller platform that supports the transition from legacy “brown field” networks to SDN “green field” networks. OpenWrt is a highly extensible GNU/Linux distribution for embedded devices. IP Infusion offers an open and disaggregated network operating system called OcNOS (Open Compute Network Operating System).  Zeek is a powerful network analysis framework. Designed for bare metal switches and white box platforms, Open Network Linux (ONL) enables organizations to develop custom network operating systems. Mininet emulates a complete network. Ryu is a component-based software defined networking framework. Floodlight is designed to work with the growing number of virtual switches. Apache MINA is a network application framework. Storm is a distributed realtime computation system. Redfish RESTful API for Data Center Hardware Management

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
